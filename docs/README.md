# <font color="darkviolet"> Geometric Lib </font>
## Общее описание решения
Библиотека позволяет работать с двумя фигурами: квадаратом и окружностью. С помощью нее можно посчитать периметр и плозадь каждой фигуры.


# <font color="violet"> Math formulas </font>
## <font color="pink"> Area </font>
- Circle: S = πR²
- Square: S = a²

## <font color="pink"> Perimeter </font>
- Circle: P = 2πR
- Square: P = 4a

# <font color="violet"> Functions </font>

## <font color="pink"> Circle </font>
### 1. perimeter()
- Вычисляет длину окружности с заданным радиусом.

        Параметры:
            r (float): радиус окружности

        Возвращаемое значение:
            perimeter (float): длина окружности радиуса r

        >>> perimeter(9)
        <<< 56.54866776461627

### 2. area()
- Вычисляет площадь круга с заданным радиусом.

        Параметры:
            r (float): радиус круга

        Возвращаемое значение:
            area (float): площадь круга радиуса r

        >>> area(9)
        <<< 254.46900494077323

## <font color="pink"> Square </font>
### 1. perimeter()
- Вычисляет периметр квадрата с заданной стороной.

        Параметры:
            a (float): длина стороны квадрата

        Возвращаемое значение:
            perimeter (float): периметр квадрата со стороной a 
            
        >>> perimeter(2)
        <<< 8

### 2. area()
- Вычисляет площадь квадрата с заданной стороной.

        Параметры:
            a (float): длина стороны квадрата

        Возвращаемое значение:
            area (float): площадь квадрата со стороной a

        >>> area(2)
        <<< 4
